In this exercise we are working with two Event t Ints which always fire at the same time.

The goal is to produce an Event t Int which fires at the same time as these inputs, where the value of the output Event is the sum of the two input Events.

You should fill out this function:

mergeWithExercise :: Reflex t
                  => Event t Int
                  -> Event t Int
                  -> Event t Int
mergeWithExercise eIn1 eIn2 =
  never

to make this happen.
