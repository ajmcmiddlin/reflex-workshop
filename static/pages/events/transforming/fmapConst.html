The goal of this exercise is to derive Events from a pair of buttons, so that:

  • the first Event will fire with a 3 whenever the first button is clicked
  • the second Event will fire with a 5 whenever the second button is clicked

You should fill out these functions:

fmapConstExercise3 :: Reflex t 
                   => Event t () 
                   -> Event t Int
fmapConstExercise3 eIn =
  never

fmapConstExercise5 :: Reflex t 
                   => Event t () 
                   -> Event t Int
fmapConstExercise5 eIn =
  never

to make this happen.

The fmapConstExercise3 function takes the Event t () corresponding to the "Click for 3" button as input, and the fmapConstExercise5 function takes the Event t () corresponding to the "Click for 5" button as input.
